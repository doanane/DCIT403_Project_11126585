COUPLING ANALYSIS
=================

Agent pair                            | Message            | Direction
--------------------------------------|--------------------|-----------
PharmacySensor -> StockMonitorAgent   | STOCK_READING      | 1-to-1
StockMonitorAgent -> SupplyAssess.    | STOCK_ALERT        | 1-to-1
SupplyAssess. -> TransferCoord.       | SHORTAGE_CLASSIFIED| 1-to-1
SupplyAssess. -> ProcurementEscal.    | SHORTAGE_CLASSIFIED| 1-to-1
TransferCoord. -> ProcurementEscal.   | TRANSFER_RESULT    | 1-to-1

Note: No agent sends a message back to a higher-level agent in the pipeline.
The only lateral message is TRANSFER_RESULT from TransferCoordinationAgent to
ProcurementEscalationAgent, which are at the same architectural level.

================================================================================

AGENT RESPONSIBILITIES SUMMARY
===============================

StockMonitorAgent
  Knows about: DrugDatabase (reads and writes stock levels)
  Talks to:    SupplyAssessmentAgent

SupplyAssessmentAgent
  Knows about: DrugDatabase (reads drug category)
  Talks to:    TransferCoordinationAgent, ProcurementEscalationAgent

TransferCoordinationAgent
  Knows about: DrugDatabase (reads all ward stocks, writes after transfer)
               WardDatabase (writes TransferRecords, reads ward list)
  Talks to:    ProcurementEscalationAgent

ProcurementEscalationAgent
  Knows about: SupplierDatabase (reads suppliers, writes ProcurementRecords)
               DrugDatabase (reads drug metadata for procurement)
  Talks to:    (none, terminal agent in the pipeline)
  Receives from: SupplyAssessmentAgent, TransferCoordinationAgent, Simulator
